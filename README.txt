This is a classic knights tour 

A knights tour is where you have a knight on a chess board, and the objective is to make a series of possible moves the knight can move to go on every single space on the chess board. 

This code was made using haskell, This code will run on many different size's of boards. form 5-9. The lowest chess board size possible is 5. If you make the board have 10 by 10 tiles then the code would need to run for years before coming with a conclusion because there are to many possibilites. 

To change the size of the board change the variable tileLengthOfBoard to the desired amount.
